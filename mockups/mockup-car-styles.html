CAR STYLE MOCKUPS

Six visual approaches — all pure canvas, no images needed

CURRENT

A — Flat Rectangles

What we have now. Simple, fast, clear.
+ Cheapest to render, very readable at small sizes
- Looks basic, all body types are just rectangles
LOW EFFORT

B — Shaped Bodies

Curved paths instead of rectangles. Wheels visible. Same render cost.
+ Distinct silhouettes per type, still very fast
- More path code per body style
MEDIUM EFFORT

C — Gradient & Metallic

Linear gradients for a painted-metal look. Specular highlights.
+ Rich, premium feel. Great color depth
- Gradient creation per frame (cacheable with offscreen canvas)
LOW EFFORT

D — Bold Outline / Comic

Thick black outlines, flat fills, punchy colors. Comic book vibe.
+ Very readable at any zoom, strong personality
- Slightly cartoonier tone
MEDIUM EFFORT

E — Pixel Art

Chunky retro pixels. Nostalgic arcade feel.
+ Unique retro aesthetic, great for the genre
- Harder to add new body styles, pixel grids need care
HIGHER EFFORT

F — Pseudo-3D Shaded

Top-down with height shading, cast shadows, roof bevel. Looks 3D.
+ Most visually impressive, professional look
- Most complex drawing code, heavier per-frame
